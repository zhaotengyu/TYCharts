# TYCharts

iOS股票分时、k线图表库，支持MA、BOLL、KDJ、MACD、RSI、WR等技术指标，支持更多指标扩展，后续可能会更新更多功能，喜欢的欢饮star一下，有任何建议或者问题可以提出来，我会尽可能的解决。

### 集成方法
1. 下载Demo项目，将TYCharts文件夹拖入需要的工程中。
2. 参考Demo项目，蜡烛图使用 TYCandleStickView 创建。
3. 其他指标视图可以使用类型创建或者使用具体的类视图进行创建。
